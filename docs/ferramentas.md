# HTML : Quais elementos estão sendo usados?
eu utilizei do H1 ao H6,para o título e subtítulo
para os textos que aparecem na página eu utilizei a tag <P>
para a criação do meu menu eu utilizei a tag <nav> e para ajudar na estilização do menu eu usei a <navbar>
para a organização das paginas eu usei a tag <div>
usei a tag <img> para adicionar as imagens
para deixar o menu organizado um do lado do outro eu utilizei a tag <ul>
para a organização eu tembém utilizei a tag <li> fazendo a lista ordenada do conteúdo do site
e utilizei o <href> para colocar o link e conectar o css

# CSS: Quais propriedades estão sendo usadas?

o css em si eu utilizei para a estilização do site
o background-color eu utilizei na propriedade do body

na propriedade do título eu utilizei o text-align: center; e o color para centratilzar o titulo e modificar a cor
na propriedade formulario-campo utilizei: align-items: center; para centralizar os textos das paginas, width: 25%; para o tamanho da fonte
